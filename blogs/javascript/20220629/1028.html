    JadeXusq 2022/6/29 javascriptnavigator

    # 简介

    1）媒体会话 API 提供了一种自定义媒体通知的方法。它通过提供元数据以供用户代理显示您的 Web 应用程序正在播放的媒体来实现这一点。 2）它还提供操作处理程序，浏览器可以使用这些处理程序来访问平台媒体键，例如键盘、耳机、遥控器上的硬件键，以及通知区域和移动设备锁定屏幕上的软件键。因此，即使不查看网页，您也可以通过设备无缝控制网络提供的媒体。 3）其目的是让用户知道正在播放什么并对其进行控制，而无需打开启动它的特定页面。为了能够支持媒体会话 API，浏览器首先需要一种机制来访问和控制操作系统级别的媒体控件

    # 使用介绍

    # MediaMetaData对象

    允许网站为正在播放的媒体向平台 UI 提供丰富的元数据。此元数据包括标题、艺术家（创作者）姓名、专辑（收藏）和艺术品。该平台可以在媒体中心、通知、设备锁屏等中显示此元数据。

    if ('mediaSession' in navigator) {
  navigator.mediaSession.metadata = new MediaMetadata({
    title: 'Unforgettable',
    artist: 'Nat King Cole',

    # 属性

    1）onLine: 网络在线离线状态。 2）connection: 网络连接状态。 3）pdfViewerEnabled: 指示浏览器在导航到 PDF 文件时是否支持内联显示。

    # 方法

    1）share: 调用当前平台的本地共享机制。 2）getBattery: 返回有关电池充电状态的信息。 3）vibrate: 在支持振动的设备上引起振动，如果不支持振动,则不做任何事情。
